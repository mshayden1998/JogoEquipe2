A Equipe 2 é formada por: André Martins, Cindi Saicosque, Joacir Marconcini e Marcelo Hayden.

O jogo é sobre um tronco de árvore humanóide que precisa usar as plataformas para não ser atingido pelas serras localizadas acima e abaixo da tela.

Para jogar, é só usar o W, A, S e D ou as setinhas do teclado.

Ferramentas externas usadas:
- edição de imagem: GIMP, www.remove.bg
- criação de vídeos: Clipchamp
- download de áudios livres: www.freesound.org, www.opengameart.org
- edição de áudios: Audacity
- conversação de arquivos: www.convertio.co
- organização de projeto: GitHub Desktop, www.trello.com
